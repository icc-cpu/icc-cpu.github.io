Dart是一种面向对象的语言，支持基于mixin的继承。每个对象都是一个类的实例，所有类都继承自Object。 基于Mixin的继承意味着虽然每个类（除了Object）只有一个超类，一个类可以继承自多个父类（多继承）。

#### 使用构造函数

构造函数可以是ClassName或者ClassName.identifier。例如：

```
var p1 = Point(2, 2);
var p2 = Point.fromJson({'x': 1, 'y': 2});
var p3 = new Point(3, 3);
```

其中的new关键字是可选的。
